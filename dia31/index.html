Eventos Javascript

1. click

Se dispara cuando el usuario hace clic en un elemento. Es uno de los eventos más comunes y se utiliza para detectar interacciones con botones, enlaces y otros elementos interactivos.

elemento.addEventListener('click', function() { ... });

2. dblclick

Se activa cuando el usuario hace doble clic en un elemento. Útil para implementar acciones alternativas o avanzadas.

elemento.addEventListener('dblclick', function() { ... });

3. mouseenter

Se dispara cuando el puntero del ratón entra en un elemento. A diferencia de mouseover, no se dispara cuando el puntero se mueve sobre elementos hijos.

elemento.addEventListener('mouseenter', function() { ... });

4. mouseleave

Se activa cuando el puntero del ratón sale de un elemento. Es el complemento de mouseenter y es útil para efectos hover.

elemento.addEventListener('mouseleave', function() { ... });

5. keydown

Se dispara cuando el usuario presiona una tecla. Este evento se activa repetidamente mientras se mantiene presionada la tecla.

document.addEventListener('keydown', function(event) { ... });

6. keyup

Se dispara cuando el usuario suelta una tecla previamente presionada. Útil para detectar cuando ha finalizado la entrada de teclado.

document.addEventListener('keyup', function(event) { ... });

7. submit

Se activa cuando se envía un formulario. Es fundamental para validar datos antes de enviarlos al servidor o para manejar envíos de formularios mediante AJAX.

formulario.addEventListener('submit', function(event) { ... });

8. load

Se dispara cuando un recurso y sus dependencias han terminado de cargar. Comúnmente usado con el objeto window para ejecutar código después de que la página esté completamente cargada.

window.addEventListener('load', function() { ... });

9. DOMContentLoaded

Se activa cuando el documento HTML ha sido completamente cargado y analizado, sin esperar a que se carguen hojas de estilo, imágenes y subframes. Más rápido que el evento load.

document.addEventListener('DOMContentLoaded', function() { ... });

10. resize

Se dispara cuando se cambia el tamaño de la ventana del navegador. Útil para diseños responsive y para ajustar elementos según el tamaño de la ventana.

window.addEventListener('resize', function() { ... });

11. scroll

Se activa cuando el usuario desplaza el contenido de un elemento o la ventana. Se utiliza para efectos de parallax, cargas infinitas o para mostrar/ocultar elementos según la posición de desplazamiento.

window.addEventListener('scroll', function() { ... });

12. change

Se dispara cuando el valor de un elemento <input>, <select> o <textarea> cambia y pierde el foco. Útil para validar o procesar entradas de usuario.

input.addEventListener('change', function() { ... });

13. input

Similar a change, pero se dispara inmediatamente cuando el valor de un elemento cambia, sin esperar a que pierda el foco. Ideal para validación en tiempo real.

input.addEventListener('input', function() { ... });

14. focus

Se activa cuando un elemento recibe el foco, ya sea por clic del usuario o navegación con teclado (Tab). Útil para mostrar sugerencias o cambiar estilos.

elemento.addEventListener('focus', function() { ... });

15. blur

Se dispara cuando un elemento pierde el foco. Complementario al evento focus y útil para validaciones cuando el usuario termina de interactuar con un campo.

elemento.addEventListener('blur', function() { ... });

16. contextmenu

Se activa cuando el usuario hace clic derecho en un elemento para abrir el menú contextual. Puede usarse para crear menús contextuales personalizados.

elemento.addEventListener('contextmenu', function(event) { ... });

17. touchstart

Se dispara cuando un dedo toca la pantalla en dispositivos táctiles. Es parte de los eventos touch y es fundamental para aplicaciones móviles.

elemento.addEventListener('touchstart', function(event) { ... });

18. touchend

Se activa cuando un dedo se levanta de la pantalla en dispositivos táctiles. Complementario a touchstart y usado para detectar gestos completos.

elemento.addEventListener('touchend', function(event) { ... });

19. dragstart

Se dispara cuando el usuario comienza a arrastrar un elemento. Es parte del API de Drag and Drop y permite implementar funcionalidad de arrastrar y soltar.

elemento.addEventListener('dragstart', function(event) { ... });

20. drop

Se activa cuando un elemento arrastrado se suelta en un objetivo válido. Complementa dragstart y otros eventos de arrastre para completar operaciones de arrastrar y soltar.

elemento.addEventListener('drop', function(event) { ... });
